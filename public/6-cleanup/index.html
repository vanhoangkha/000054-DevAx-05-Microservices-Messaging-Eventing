Clean up resources

You clean up resources in the following order:

Terminate EC2 Instance

  1. Go to Amazon EC2 console.
  • On the left navigation bar, click Intances.
  • Select the instances we created in this lab.
  • Click Instance state
  • Click Terminate instance
  1. Click Terminate

Delete Users

  1. Go to AWS IAM Console.
  • Click Users.
  • Type awsstudent to the search bar
  • Select awsstudent.
  • Click Delete
  1. Type awsstudent to confirm, then click Delete

Delete CloudFormation Stack

  1. Go to AWS CloudFormation Console.
  • Select DevAx-05.
  • Click Delete
  1. Click Delete stack

Delete Lambda function

  1. Go to AWS Lambda console.
  • Click Functions.
  • Select Kinesissam-LambdaFunction.
  • Click Delete
  1. Type delete to confirm, then click Delete to delete

Delete CodeCommit Repositories

  1. Go to AWS CodeCommit console.
  • Click Repositories
  • Select KinesisProducer
  • Click Delete repository
  1. Type delete to confirm, then click Delete to delete
  2. Do the same for the other CodeCommit Repository

Delete S3 bucket

  1. Go to AWS S3 Console.
  • Click Buckets
  • Select first S3 bucket.
  • Click Empty.
  1. Type permanently delete to confirm, then click Empty to delete the data of this S3 bucket.
  2. Click Exit to back to the S3 inteface
  3. Click Delete.
  4. Type the name of the bucket then click Delete bucket to delete S3 bucket.
  5. Do the same for the other S3 buckets
